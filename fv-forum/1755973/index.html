Pencil to allow breaking the chain?

  • I'm going to try out FV today on paper, using my existing list. However, I'm nervous about getting stuck with a set of preselect dots after a period of work. If I need to 'break the chain' due to circumstances, I don't want to leave the dots hanging, and I don't want to cross off and reenter the undone tasks at the end. So I will use a pencil in the preselect phase, and reinforce the current task with a penned dot.

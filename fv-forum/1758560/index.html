  • My interpretation (which might be wrong) is that 3 is your "last preselected task" because it's further down on your main list. Here's my chain of thinking:

    "Circumstances have changed. It's still true that I must take action on task 1. I've already done some preselection, so it's probably still true that I want to take action on task 3 before I take action on task 1. However, is there anything new that has come up that I need to take action on before I take action on task 3?"

    If this is the correct way to interpret it, then your dots stay relevant. Any other interpretation that I can think of requires erasing dots or ignoring dots or rewriting tasks to get rid of dots.

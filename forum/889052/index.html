AF4 done wrong is good too

  • I'm a big fan of Mark's suggestion to try the instructions as written for a while -- before coming up with a list of all the reasons the system won't work. So I read through the AF4 instructions, watched the PDF demo, and struck out on my own for a week.

    Scenario: I started with my 47 page list built from AF1 onwards. I'm uncomfortable with the length of the list, but I'm comfortable with my intuition telling me when something needs to be done. The week was extremely productive, and I did (or crossed off) a lot of backlog items. And I'm down to 40 pages. But...

    I've been operating the system backwards. I make one pass through the closed list, then loop through the open list doing doing doing until nothing stands out. Then I return to make a single pass through the closed list. This procedure made sense to me during the week. Give more attention to the recently-added/possibly urgent stuff and less attention to the backlog. Now I have to go away again for a week and operate the system as written to see how that goes.

    My only question is... should I read the discussions now? Maybe they hold vital info... or maybe they'd help me form preconceptions before I get experience. Ayiieee.

    In any case, it must be a great productivity system that works so well even if you do it wrong. Thanks Mark.

  • Slothbear:

    I think it's a bit much to expect any new system to be able to cope with forty-seven pages of backlog. I've decided to change the Backlog name to Closed List precisely because I didn't want that to happen.

    AF4 is designed to process your active work. If you have stuff which hasn't been touched since February, then by no stretch of the imagination can that be called active work.

    I would strongly advise you to start off a fresh list. Just write a short list of the things you are currently working on, and add to it as things come up in the course of work.

I'm really getting a lot done, and finishing off that closed list is finally in sight. And at least half of the tasks on the Open List have been completed.

    AF3 was "faster" as far as responding to urgent things. But urgency began to completely dominate, at the expense of importance.

    With AF4, it's still pretty responsive to urgency, but the focus on importance has returned.

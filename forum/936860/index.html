Another AF4 and / or DIT iphone app ~ and this one IS free

  • Hi All,

    Just rooting round the App Store and came across ZenTask.

    This is a free iphone app companion to a Mac app of the same name.

    Very simple clear interface which just lists new items in the order you enter them.

    Except that on the next day a new list starts at the top. Apart from today's new entries tasks remain listed in ascending order of entry.

    Completed tasks are filed separately so the working list stays clear.

    Tasks can be grouped by due date and there are automatic groupings for Today, Tomorrow, Past Due and All Tasks

    For DIT you would always set Due to tomorrow, then work on the Today list when the time came around.

    For AF4 you would work on All Tasks

    It is also possible to set any other grouping for projects / locations, forward due dates, pririties and recycle taks if you wanted to use them.

    The app syncs to Mac desktop app if you are a user but I have not tried this as I am still working on PC.

    Interesting App able to run various MF task philosophies with scope to tweak if you want more features

    Cheers

  • About the (wonderful, useful) userscript: on Firefox, anyway, I was able to get it working again without much trouble. Here's what I did:

    - Go to Tools/ Greasemonkey/ Manage User Scripts....
    - The Add-ons Manager window will open, showing your Greasemonkey scripts
    - For markforster.net improvements 2, click on Options
    - On the User Settings tab, next to Included Pages, click Add
    - In the resulting dialogue box, type http://markforster.squarespace.com/* (sorry that comes out looking like a clickable URL--again, it's just the text you should type) and click Okay
    - Back at the User Settings tab, click Okay

    After that, when you return to Mark's site using the URL that includes "squarespace," the user script should work.

    Hope that helps.

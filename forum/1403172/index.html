Startup confusion

  • I'm Starting a new list and I have only part of one page so far. I think the rules ought to be clarified for the startup phase.

    As written, it seems like I should reenter stuff on page 2, but I think that's incorrect and the "next page" is actually page 1. Confusing.

    Following the rules, I may write a few things down, alternate between a couple, add an urgent item, etc., and completely fill column 2 before I even come up with 30 nonurgent tasks. Now I'm in trouble.

    As long as only one page exists, there's little point fretting over whether something is active, urgent, unfinished, reentered because all tasks on the page are available. So I'd suggest to simplify, everything goes column 1 until I have a second page started.

    Yet I shall attempt to abide the rules as written.

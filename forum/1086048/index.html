DWM.xls - New version

  • Hi,
    I posted a new version of DWM.xls

    There are many changes:

    - Configuration worksheet: Check the different options (time/date formats, colors, amount of days)

    - New column "Entered" (different from "Created", as it's updated on every reentry). It comes to fix a bug reported by Mark (Tasks will be sorted by this column instead of “Created”)

    - Import tasks from old file: No need to copy/paste from your old file anymore. It will only work with files formatted for the old version (without additions, etc).

    - Closing Date removed (it duplicated "End Time")


    There's quite a bit of programming. I hope there aren't too many bugs.
    Let me know what you think.

AutofocusAHK 0.9.5 release

  • Hi everyone, version 0.9.5 is here: http://github.com/downloads/andreashofmann/AutofocusAHK/AutofocusAHK-0.9.5.zip

    New features:

    - Tickler functionality: Add tasks at a given date
    - Auto-copy selected text when adding a new task
    - Added checkbox to confirm auto-copied URLs

    Tweaks:

    - Added a button to the work window to mark tasks as already done without actioning them
    - Added a button to the status window to immediately re-add the current task
    - Export: Don't show headers if a section hasn't got visible tasks
    - Export: Only show needed columns when done/review tasks are hidden
    - Export: No option to show tasks on review if current system is DWM
    - DWM: Show expiration date in title of work window (thanks to sabre23t)
    - DWM: Work window changes color when current task expires today/tomorrow

    Fixed bugs:

    - Clipboard content is no longer lost when auto-adding URLs
    - Task expiration pop up now lists unactioned expired tasks only

  • Yes, the sorting mechanism (which seems to be broken in 0.9.5.2 even though not a single line of its code was changed, must be a concurrency issue) only sorts the newly (re-)added task in, it doesn't fix existing unordered tasks. So if you have a task that expires March 5th but is in the March 10th slot, when a new task is added, the sorting mechanism will look backwards through the list, find that task, and say "ah, expires on March 5th, the new tasks goes after this one". And so the new task will be listed under March 10th, too.

    Version 0.9.5.3 will perform a complete resort of the list to fix this, so you don't have to sort the list by hand now if you don't want to. I promise 0.9.5.3 will be released this weekend, before any of the affected tasks get in danger of expiring. The right order will then be restored automatically.

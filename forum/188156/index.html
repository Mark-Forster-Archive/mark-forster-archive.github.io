  • Hi Buzz, I used to be a real Outlook fiend but I found that it was way too easy to generate a lot of tasks which then felt overwhelming. I still use follow up flags on emails if they need to be actioned and when they are completed I fill in the followup flag with the result and mark them as done. As I work at the computer all day I don't feel like using it when I go home so using Outlook for personal tasks too is not something I find emotionally satisfying. I have gone back to the old paper and pen method of using a pukka project book which has 3 sections. The first section I use as a journal, the second section is for "thoughtdump" - anything about projects/working stuff out that needs to be done. The third section I use for goals so I know in what general direction I am supposed to be headed in. There is something really satifying about crossing things out and writing notes to yourself by hand as well as celebrating your achievements by writing down milestones along the way. I also use stickers a lot to make things look pretty! You can't do that in Outlook! This works very well for me but I know it wouldn't suit everyone. For reminders I just use my mobile phone to set up things I need to remember and that is really handy.

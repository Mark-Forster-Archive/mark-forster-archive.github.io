  • Hello all, I've followed this forum and the blog for a long time (since DIT). But there are some issues that have kept me from jumping in. So now I've decided to put them all out there one by one and to see if I can make this work for me.

  • Lynn,

    Returning to your original post, in AF1 Mark advised users to carry a pocket-sized book when AWAY from the "formal" AF list.

    One of the reasons I favoured A5 sized notebooks when I was 100% manual was because an A5 size book can fit into most pockets, briefcases etc. A6 can be used for when one is away from the formal A5 notebook.

    Nowadays I use a combo system: ListPro (on computer and PDA/Windows Phone, synchronised). I print out daily the ListPro AF4 list sectioned into Daily Routine, Backlog/Closed DIT, Backlog/Closed non-DIT, and Current/Open.

    At the start of each day, I print the ListPro list. New or started, but uncompleted tasks from the ListPro list are written in AFTER the end of the list. Completed tasks are crossed out, as are any written tasks if they are tackled. Thus after printing, I am essentially back in manual mode although I also "tick/delete" tackled tasks n ListPro as well if I remember

    At the end of the day, I do the following:

    (1) Cross out in ListPro all tackled tasks EXCEPT for Daily Routine tasks ie any crossed out electronically, these are restored as unticked for the next day
    (2) Any UNtackled tasks written down are typed into ListPro ready for the next day.
    (3) When the time is ready for review in terms of AF4, I can change the status of tasks very quickly eg Current/Open tasks could become Backlog/Closed.

    The next day I start again by printing the ListPro list (updated) again.

    So my daily AF list is ONE A4 sheet, with the ListPro list printed as 2 x A5 pages on one side.

    BUT, I also carry an A5 or similar-sized Moleskine most of the time in which to write notes, doodling etc, with the day's AF4 list folded in half and tucked into the A5/Moleskine notebook.

    Works for me.

  • What Alan said. My AF book IS my capture tool. Once an item hits it, then I decide if it is:

    - a "someday/maybe" in which case it gets "yellow lined" right then;

  • Each Post does have a number, but you don't see it normaly. You have to right-click on some whitespace on the page and select "View Source" (This may vary depending on which browser you use). Then find the specific post you want to link to, best by searching for some of its content or the poster's name. The post is surrounded by several HTML tags, and in one of them you'll find something like this: id="post886075"

    That is the post number. To link to this conversation, you'd use this link: http://www.markforster.net/forum/post/885566

    To link to this conversation and specifically the post where you asked this question, you'd use this: http://www.markforster.net/forum/post/885566#post886075

    I know that's not really comfortable and doesn't help you to refer to a post in conversation with someone, but it works for linking to specific posts.

    Re knowing where you were last time, the SiteDelta addon for Firefox is somewhat useful for that. I posted a demonstration for it here: http://www.markforster.net/forum/post/850591#post852041

    Recently I stopped using SiteDelta, I'm programming my own "forum reader" to get around some of its shortcomings. I intended it for personal use only, bu if Mark has nothing against it, I could make it available for public use.

  • I attend (and run) a lot of meetings, and someone has to capture the minutes and the action items. Often it's me. I usually write those up in an email which I send out at the end of the meeting, to all the participants and all the need-to-knows.

    While taking the minutes, if I myself get an action item, I'll add it to my AF list right then.

    If I don't have time to do it right then, then I'll scan the action items after the meeting, and copy mine to the AF list. Also, if someone else has action items but I need to track their progress for some reason, I'll add that to AF too ("follow up with Mike on the new XSL").

    This is all very low overhead, and works very well. It works well even if the meeting is just a 1-on-1, or just my own personal project-organization session.

    If I'm not sure if something is an action item or just a "note", I'll add it to AF and let AF decide. :-)

  • Hi Mary,

    I'm using "standard" Autofocus. I have two notebooks, one for work, one for personal. Just regular cheap stitch-bound notebooks with about 100 pages each. For work I like using a graph-paper format, for home I use a college-ruled book.

    I think the only "exception" that I make to the rules is when I have a sudden batch of very urgent tasks ("must do today"). In that case, I add the tasks to the last active page, as usual, but then make that page my "active" page and start working it as normal. The only rule I am breaking by doing this is not going through the pages sequentially, but jumping straight to the last page.

    Otherwise I just use standard AF.

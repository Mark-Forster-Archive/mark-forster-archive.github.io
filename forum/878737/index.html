  • Great question. I am not super web savvy so I will try to outline it here.

    In a nutshell, the Ultra Simple Guide to Time Management is a 10 page pdf file which outlines the time management method I have been using successfully over the years. Some of you may remember my first ebook which was the Simple Guide to Time Management which was a spiral notebook based system. The new system is intended to further simplify and enhance that system be a complete time management system which addresses what I feel is an issue with many systems namely the integration of project management and task management. Many of us need more than a task management system, we need a way to integrate some level of project management into what we do. The system is paper based and designed to be that way on purpose

    The main tool in the system is a simple legal pad, like almost any office has. The method includes a concept I call Rigid Flexibility, which is the manner in which the legal pad is used to maximize efficiency and effectiveness. There are a very few rules to the system, but the ones that are there are very key to it working. The system is easy to start and keep up and is the one I have used successfully for the last decade.

    I am happy to answer any questions about it here or on my discussion forum, which is linked from my blog.

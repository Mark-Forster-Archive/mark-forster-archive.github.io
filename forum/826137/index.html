CONDENSED AF2 anyone?

  • In the last weeks, I developped the habit of marking tasks as "I should really start this one as soon as possible" while reading through the whole list - similar to how it was later proposed as CONDENSED AF. I am asking myself whether this is a useful technique or not.

    Pro:

    + The very thought of "this one is ripe to be acted upon" might be considered as coming from the unconscious and therefore worth an action, and be it a simple mark besides the task.

    + If you jump from marked task to marked task, you make your way through the list more quickly.

    Contra:

    - Over time, more and more tasks get marked the same way (and not necessarily acted upon the same day), so the value of that mark diminishes. - Question: Is this really a bad thing? And what about repeating the mark? Why not having !!tasks and !!!!tasks?)

    - Marking reflects the state of consciousness in the moment I read through the list, but not necessarily the state of consciousness in the moment I have to choose the next tasks. So it may bias the "standing out" process.

    What do others think?

  • Andreas,

    I think that any single tweak might be very useful. I also think as tweaks tend to be added to tweaks you might end up with something really tangled and convoluted.

    I also have experimented with marking priority/urgency/importance for tasks. It changes the character of the AF list idea. One thing I noticed was that I marked TOO MANY things as high priority. In a sense, EVERYthing on my list is a high priority for me! ;-) But if you have more than one high priority item, then you probably don't have a good handle on what "priority" means. That was what I concluded for myself, anyway.

    My solution was to pull items off the list onto an index card of things I really intended to do TODAY. It gave me a view of what I had on my plate and things were not left buried in a long list. (Marking a priority is not much good if you don't even see the task!)

    Basically, the way I view it is that if you have multiple tags, priorities, colors, symbols, Post-Its, paper clips, etc. you are probably playing with your list way too much and should probably think about DOING some of those things ;-)

Beta testers wanted for FocusPad, an iPhone app for AF

  • Hi all.

    I know that many of you are hungry for an iPhone app to use with AF. There are some out there that work okay for this, but they all have their shortcomings.

    The good news is that I have created an iPhone app called FocusPad that is ideally suited for using with AF. It is page-oriented and allows tasks to be easily removed, highlighted, or moved to the end. You can see some screenshots at:

    http://blog.7actionsdev.com/2009/08/focuspad.html

    I am looking for beta testers. If you are interested in being a tester, email me at tony@7actions.com.

  • Mel,

    To mark a task completed, you "delete" it, which grays it out and leaves the task disabled from further operations. Moving a task to the end deletes the task at its current position and creates a copy of it at the end of the last page (unless the task is already on the last page, in which case it just moves it). This is in line with the AF philosophy.

    To move pages, you swipe left or right, or use the slider for larger moves. It feels very good.

    There is currently no way to move between highlighted tasks. However, at some point I would like to add a page thumbnail browser, which would let you spot highlighted tasks quickly.

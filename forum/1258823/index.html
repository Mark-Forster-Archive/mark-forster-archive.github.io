Trying to adapt DWM (AF5) as AF4 - Following project digital

  • There is something I smuggle since a few month ie following project and context

    Yes AF4 is a wonderful method. Because it makes getting things done
    DWM is not silly as well because it obliges yourself to put deadlines about thing and dismiss naturally unusefull stuff

    But what's about following in a glance all tasks about a project ?
    What's about contexts ? I need to do all my calls and emails at a batch.
    When I shall buy an ipad I want to have my AF4 list if I can even if I love paper. How can I do it.

    I may have found a solution :
    I am still testing but I wanted to share it with you immediately ;-)

    1. I use numbers I am a mac user
    2. I Created 4 colums
    Contexts = Wait, Call, Mac, Work
    Tasks = What i have to do and who is concerned
    ie Mr smith 01 55 25 36 XX / send proposal
    Projects = project's name ie BUILDING XX
    Deadline = deadline with DWM method (new tasks =+1 month, reported = +7 days)
    Dashboard for the date

    The list is classified by deadline

    2 tips. (exception about the rules)
    If something must be done a certain date i put it on.
    If I need to do something in a week i put it on

    If i need to classified by category (project, context, date) i just click on.numbers column

    Advantage
    it goes very very fast fast
    it's easy
    The control is fantastic
    it make me see all in a glance
    I will have it on my ipad then anywhere
    I can do it on excel too but i prefer number for this
    I have a strick following of tasks like on AF4 that i love (the list is naturally classified little by little)
    I dont need to read it again and again just a quick glance of all
    I can erase immediately what is unusfull
    I can add contexts ex dismiss ?
    I can drop anything which happen on the flow without thinking i will see it later
    The systems is regulated by itself.

    Disadvantage
    I must use a digital solution, it's better than on paper

    What do you think about it ?

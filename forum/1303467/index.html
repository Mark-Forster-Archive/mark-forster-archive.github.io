Rules for Super AutoFocus

  • Normal AF1 rules, except:

    1. There is a second column on each page which is used for:
    a. Re-entries of unfinished tasks;
    b. Tasks not already on the page which you wish to do now. (Note 1)

    2. The 2nd column is treated as part of the page for all purposes, including dismissal.

    3. If there is insufficient room in the 2nd column for more tasks, then any further tasks for it are entered in the 2nd column of the next available page. (Note 2)

    4. If a task in the 2nd column is the only task ready to be done, it should be crossed out and moved to the 2nd column of the next available page (Note 3). If no other tasks have been done on the current visit to the page, then all other tasks should be dismissed including any remaining in the 2nd column. (Note 4)

    Notes:

    1) This is intended to be used for anything you want to do now, not just urgent tasks.

    2) This imposes a limit on how long new material can be added to a page.

    3) The effect of this is to allow one task to continue to be worked on for more than one page.

    4) This prevents the addition of a task to a page solely in order to avoid the page being dismissed. Rule 4 as a whole also prevents a task being worked on twice running without another task intervening.

  • Seraphim:

    << I'm trying to understand Rule 4 >>

    I'm sorry it's difficult to understand. Writing clear rules is surprisingly difficult.

    Basically Column 2 is just an extension to the page it is on. It is a way of putting more tasks onto a page that is already full.

    It is particularly useful when you have an unfinished task. Instead of having to re-enter it at the end of the list, which may be a long way off, you can put it back on the same page so that it comes round again very quickly.

    A problem however occurs when all the other tasks on the page have either been completed or you have done as much work as you want to on them for the time being. In that case the unfinished task is the only one available to work on and you can't alternate it with any other task.

    This is where Rule 4 comes in. In this situation you can move the unfinished task to the next page instead of to the end of the list. There it can alternate with some fresh tasks without losing momentum.

    It's possible with a really big task that it may continue moving round the list for some considerable time.

    The final bit of Rule 4 is for the situation when you move the task to the next page as above, but when you get there you find none of the tasks already on that page are standing out. In that case the page would have been dismissed anyway, so that is what you do.

    Hope that's clearer!

  • Mark,

    Thanks for the explanation of Rule #4! I think I understand it better now. I'll try it in my own words again to see if I got it:

    (1) If you are working on a task, and then decide to stop, and the task is unfinished, you then write the task at the end of Column 2. You then cycle through all the tasks on the page (both columns 1 and 2), looking for something to "stand out". If you cycle through both columns, and nothing stands out, and then you arrive at the task you just wrote at the end of Column 2, you should cross off that task, move to the next page, and add that task to the end of Column 2 on that page (carrying along this task with you as you go forward through the AF book).

    (2) Then, following the AF rules, you quickly read through all the tasks on the new page (both columns) to refresh your memory on what's there. You then start reading through them more slowly, looking for one to stand out. If you go through all of them, and nothing stands out, and finally arrive at the task you just entered at the end of Column 2, then you should cross out the task you had just entered at the end of Column 2, dismiss everything on that page, move to the next page, and write that task again at the end of Column 2 on that page. Then repeat this process (2).


    The effect is that you can carry forward one task along with you from one page to the next as you work through the AF book. This enables you to keep momentum on some particular task that keeps grabbing your attention, while still working through the whole AF book and taking care of whatever else also stands out. These particular rules also ensure that this carrying-forward doesn't interfere with the AF dismissal function.

  • Hi,
    It took me some time but here's the new version.

    Basically, the new functionality involves "activating" a task. This is an alternative to using the stopwatch.
    Upon activation, the starting time is registered, and when the task is completed, the ending time is also registered and the invested time is calculated.

    An activated task is displayed with a yellow background.
    When completing or re-entering, the active task will be selected, not the one where the cursor is placed.

    New shortcuts:
    Ctrl-Shift-A: Activate/deactivate a task
    Ctrl-Shift-J: Jump to the active task

    Please notice that there are new columns in the "Open Tasks" and "Closed Tasks" worksheets, so you'll have to re-arrange your tasks accordingly.

    To do this, there are a few steps involved:

    1. Press Ctrl-Shift-Z: This disables some automatic functionality that would mess things up when copying and pasting.
    2. Copy your tasks, and rearrange the columns
    3. Press Ctrl-Shift-Z to restore functionality.

    There are a lot of changes in this version, so I suggest to keep a backup copy.
    Please let me know of any bug.

    Download it from here:
    http://rapidshare.com/files/372612499/DWM.xls

  • Hi Mark,

    It's actually pretty simple. You enter the text of your tasks on the "Description" column, press Enter, and the expiration is automatically computed.

    After your list is built, the basic functions are:
    Ctrl-Shift-C: Mark a task as complete. It's actually moved to the "Closed Tasks" worksheet.
    Ctrl-Shift-R: Reenters a task, changing it's expiration to today's date + 7

    Tasks that are about to expired tomorrow will be shown in red, and the expiration will be performed automatically (they are moved to the "Expired Tasks" worksheet).

    This is the full list of commands (some of them explained by Moises above)


    Ctrl + Shift + A: Activate/Deactivate task (start working on it)
    Ctrl + Shift +C: Complete
    Ctrl + Shift + D: Delete
    Ctrl + Shift + J: Jump to active task
    Ctrl + Shift + H: Highlight/Dehighlight task
    Ctrl + Shift + N: New
    Ctrl + Shift + P: Print
    Ctrl + Shift + R: Reenter
    Ctrl + Shift + S: Sort tasks
    Ctrl + Shift + W: Work on a specific task (similar to Ctrl-Shift-A, but with a stopwatch)
    Ctrl + Shift + Z: Events On/Off

    I think that I'm going to write a short guide and post it here.

  • Hi, Mark

    For starters, I'm glad that you find it useful, and of course you can post it on your site.

    About the bug, I see what you mean. I didn't think about it.

    The explanation for this is that each task keeps their creation date/time also after reentering.
    Then, if newer tasks fell under the same expiration date as the reentered one, they will be shown after it, because their creation date is newer.

    As I wrote in the guide, this is on purpose. This way if lets say, I worked four times on a specific task, I can know how much total time I invested on it.

    There are two workarounds for this:

    - Keep both a “Created” column (fixed between sessions) and a new “Entered” column (that varies after reentering, and good for tracking).

    - Create a new column, “Task Number”, generate it automatically, and keep it between sessions.

    I'd appreciate to hear some feedback from the users about this.

5 speeds

  • SF admits 5 speeds of operation for tasks.
    Now(1min) - just do it.
    Urgent(1h)- enter C2, do it before turning page
    Soon (1d) - enter last page, do when you arrive
    Near Future (1w)- enter last page, do sometime before page is dismissed
    Medium Future. (1mo) - enter last page, let it be dismissed, resurrect upon a future review
    parked (Never) - enter whereever, delete any time.

    Also for recurring tasks and big tasks, you can work them at any of the above frequencies, with the C2 option allowing a task to be picked up hourly for a while then set aside for a day or so.

Resistance/Friction starting on the first page in the morning

  • One thing that I notice with AF is that when I start the day, because the first pages are items I've been postponing doing, I find some friction right there getting going. One obvious workaround is to start on the last page and then work up to the earlier pages. Anyone else finding this?

  • I would suggest to not start at the first page in the morning, but continue right where you left off the other day. This way you get to tackle the dreaded early tasks at different times each day. Perhaps they won't be so bad after lunch.

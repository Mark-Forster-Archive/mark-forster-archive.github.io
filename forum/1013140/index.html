  • Will means the four bookmarks one can use to mark the four significant places in the list (the first active page, the current page, the week page and the month page). In a digital implementation of dwm, the week and month page bookmarks are not needed, since the software can take care of where to (re-)add a task (some people may want to be able to quickly jump to these places anyway, to take care of new tasks). Likewise, the bookmark for the first active page is only necessary if the software doesn't remove expired tasks from the list. What is left is remembering where you currently are on the list, which Focuspad already does.

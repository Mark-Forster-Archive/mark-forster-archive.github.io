3 modes of self management?

  • I wonder if it helps to orientate one's self to operating within 3 basic modes of self management:

    Inbox Zero

    Discretionary AF (or DIT, AF 2-4R, DWN, GTD etc.)

    Schedule and Routine

    As these are modes, it is possible to move between modes several times daily as neccessary.

    I wonder if this would help keep the AF list length manageable whilst also offering a holistic approach to self management. I prefer to operate Inbox Zero once per day. But I know this is unrealistic for many.

    Sorry if this is very basic stuff. As I've said before writing posts seems good for consolidating my own thoughts/ideas.

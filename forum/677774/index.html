How I use ListPron on Palm to implement Autofocus

  • The Auto Focus system promises to help me clear my back log of ever increasing lists of things to do. After reading some posts about ListPro I thought I would share my setup to help others and get feedback.

    ListPro set up. These are the fields:

    Done (Checkbox)
    Item (String)
    Category (Category - contains values HOME OFFICE LUNCHTIME DISMISSED)
    Pageid (Category)
    DueDate (Date)
    DateAdded (Date) - set to automatically enter today's date - helps keep track of the
    age of tasks and to identify areas of procrastination

    At the end of each week I update the Pageid value to group the
    remaining items. After the first week of using the system I sorted
    the list by PageId then entered 1 into the field. After the second
    week, the list whould show 1 and blank, so the blank fields are
    updated to 2. Hopefully you get the idea!

    I have annotated the Quick Start guide from the web site to show the steps....

    The system consists of one long list of everything that you have to
    do, written in a ruled notebook (25-35 lines to a page ideal). The Listpro
    equivalent is one long list with a "Date Added" field to show the creation sequence
    The PageId field is used to logically group items into weekly
    pages. This is a manual step done each week (described later)

    As I think of new items, I add them to the end of the list.
    by entering a new item in Listpro, filling in the Item field with details and
    choosing the category (HOME for things to do at home, WORK for things
    to do at the office and LUNCHTIME for errands). The category
    effectively creates the separate lists for home and work.

    You work through the list one page at a time in the following
    manner.

    1. Read quickly through all the items on the page without taking action on any of them.
    Listpro: The current page is displayed by choosing the Filter option
    and choosing the value of blank. Otherwise work through the page
    numbers in the file.

    The category field can also be filtered if focusing on home or work.

    Go through the page more slowly looking at the items in order until one stands out for you. [Listpro - scroll using Item View or LIst view for the bigger picture]

    3. Work on that item for as long as you feel like doing so

    4. Cross the item off the list, and re-enter it at the end of the list if you haven't finished it.

    Listpro. If I have finished the task, click the Done checkbox, otherwise remove the page number to move the task to the latest page.

    5. Continue going round the same page in the same way. Don't move onto the next page until you complete a pass of the page without any item standing out.

    6. Move onto the next page and repeat the process. [Listpro - choose the next page number in the Filtered View]

    7. If you go to a page and no item stands out for you on your first
    pass through it, then all the outstanding items on that page are
    dismissed without re-entering them. (N.B. This does not apply to
    the final page, on which you are still writing items). [Manually change
    the category to DISMISSED. Alternatively I can just delete the items but then all trace
    is lost!

    8. Once you've finished with the final page, re-start at the first page that is still active.
    For Listpro, the first active page is the page with the lowest page number. Completed pages will be removed (archived) from the file.

    The Listpro approach has some benefits:

    If a task needs to be done today or on a particular day, I schedule
    the item by choosing a date in the due date field.

    Different ways to Work the list

    - Check what needs to be done today:
    Filter the view to show all categories and choose today's date.

    - Check what other things that can be done in this context (work or
    home): Filter the view to show specific category and optionally a page
    (current or previous).

    - Review a page: Filter the view to show a specific page but all
    categories.

    Weekly maintenance.

    Each week I review the list using the Desktop version of ListPro. Checked items are deleted, and the page numbers for the current week are updated to the next available number. If most items of work have been completed, this task should not take long.
    Deleted items are exported and archived.

    Multiple AF "books"

    A separate task diary is recommended for work and home, or any other
    major component of your life. I consider this equivalent to the GTD
    concept of "contexts". I use the Categroy field to identify tasks to do at the
    office, errands during lunch breaks, home, and reading. Assigning tags
    allows faster review of items.

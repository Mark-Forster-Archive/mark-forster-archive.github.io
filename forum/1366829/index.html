Software for any flavor AF

  • There is a great software package called TaskPaper that can be easily customized for any flavor of AF. I have used it, but since there isn't yet a version for Droid, I am sticking with paper. It is for Macs.

    There is an associated program (the developers talk to each other and coordinate) in Windows called TodoPaper. It's slightly behind TaskPaper, but will shortly release a version 2 update that will bring its capabilities into line.

    TaskPaper has the ability to easily enter in a task and easily tag it with any keyword you want. You can also customize it to color a task based on a tag. I used it to "flag" tasks (according to the method I wrote about in the post entitled, "Systems are losing Simplicity") and then I use the filter function to show only those flagged tasks. I then cycle through those tasks – all others are hidden. When I project is marked done, it drops from the filtered list. At the end of the day you can archive the done tasks, which will remove them from the list.

    It is a remarkable program. You can customize it to any version of AF – though I don't know why anyone would want any other system than the one I proposed! :-) And, I'm sure that once TodoPaper goes to version 2, it will be just as remarkable.

    As soon as there is a Droid version, I will use it again, because then my list will always be with me. Right now my paper version is working superbly.

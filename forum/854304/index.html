Handling pending items

  • How do you handle things like waiting for a return phone call? Several of my items are calls/email to people; once I do my part, I'm waiting for them to get back to me.

    Do you leave this untouched until it's a completely finished task, cross it off and then re-enter at the end of the list or something else?

    thank you.

  • Ann,

    I enter a new task as 'Check for reply from...' or 'Follow up... if necessary'. Then cross out the original and continue as usual.

  • I use www.memotome.com and get it to send me an e-mail in a time to remind me to check if the task is done. I started off using the list but found two things that irritated me:

    1. The task could go into review mode before I could have reasonably expected the other person to have done the task. (Thinking of waiting for items here)

    2. I only want items on my list that I can do now, not a pile of stuff to do in the future. This future stuff I just park in memotome but any tickler/bringer forward file would work.

    I prefer not to use my calander for this unless the item has to be done on a particular day. This avoids cluttering up the calender. I'd much rather get a reminder and go yep its done now or if not done say I need to follow up with X and put that down as a task on my RAF list.

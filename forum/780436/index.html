Favourite Mind Map Program (for Mac)?

  • Seems like there is a LOT of choice out there these days, and I am a little overwhelmed with what to try. So I would be very interested in hearing what *your* favourite mind map program is, and why! I experimented with CMap over the weekend, but I would like something more Mac-friendly (for example, standard Mac dialogue boxes for saving files would be less cumbersome!). I would like to be able to use the mind map program with a tablet (I think most allow that?) and being able to print selected branches would be nice. Also, I would like robust "exporting as outline" capabilities, drag and dropping of resources, robust printing options (for large mind maps). I have been looking at iMindMap and MindManager (which is what I think Mark and Christine both use). (I am intrigued by the approach of MindGenius, but don't think it is available for the mac). Something very intuitive for a Mac user (ie very short learning curve!) would also be appreciated!

    Any and all comments about what you really like about the mind mapping software you are using would be most appreciated!
    thanks in advance! - Bev

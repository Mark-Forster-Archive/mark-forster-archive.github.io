  • Version 0.9.3.2 is out: http://cloud.github.com/downloads/andreashofmann/AutofocusAHK/AutofocusAHK-0.9.3.2.zip

    Just one nice new feature, inspired by vegheadjones' excel solution: an additional url field per task. So you can save a website's address along with a task (for example, I have the task "Check Mark Forster's forum, and the url is http://markforster.net/forum).

    The neat thing: If you are in a browser (IE, Firefox, Opera or Chrome at the moment) and press Casplock+a to add a new task, the url of the website you're on is automatically copied into the url field. And if you start working on a task with a url, it gets opened automatically.

  • Mel:

    The hotkeys I use most are Capslock+a a for adding tasks, Capslock+d for doing/circling through the tasks, and Capslock+e to show the complete list as a HTML page. The latter is what you're looking for, I think. It gives you the whole list, the current task is in bold font. If you have javascript enabled, you can show or hide unactioned/done/dismissed tasks via the buttons on the top right. And the list is grouped depending on your selected system (by page for AF1/AF3. by closed and open list for AF4, by expiration date for DWM). If you use DWM (remember, you can always switch the system you use in the preferences), tasks that expire today are marked red, tasks that expire tomorrow are marked yellow.

    I originally intended for Capslock+a,s,d to be the main keys, but the "show next tasks" funktion on Capslock+s is rather useless compared to the HTML export. I plan to improve it in time, for example by enabling quick jump.

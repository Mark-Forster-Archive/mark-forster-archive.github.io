Autofocus options for the IPhone

  • I've thought about a number of options for using this system on my IPhone (which I'm addicted to). I thought about using Things, RTM, Toodledo, or ToDo and tagging the tasks, but that seems too complicated. A better option is Magicpad which is a Notes app ($1.99) for the IPhone that allows you to copy and paste, strike out and color text. I like it, but for quick entry of tasks, it isn't great. Plus, there isn't a desktop or online application to sync with. I think I've found the best option (for me at least!). It's www.tadalist.com. It's free and it has an IPhone web app. This has the disad of not working without internet access and being a bit slow away from wi-fi, BUT it's easy and works beautifully with this system. Create lists called Page 1, Page 2, etc. Unfortunately, they are ordered from most recent to oldest. You can easily see how many items are in your list so you can keep it to your prescribed number. When they're completed, they're crossed off and put at the bottom of the list. That may or may not suit you. There's no real way to highlist dismissed tasks, but I realized I can simply add an @ sign before each dismissed task or use another notation. Finally, when a page is no longer active, you can simply change the Page name to "Page 1 - inactive." When you look at your lists, you'll see the most recent active page. It's FREE! And you can email lists, too.

  • Hi Mel,

    I'm also an iPhone addict and do most of my reading, writing and processing on computer or iPhone rather than paper.

    I'm intrigued by your system. However, like Mark Forster, I still prefer paper lists. For now I'm trying paper first. For me, paper is faster to write and easier to read and annotate. It's messier too but that seems to matter less for AF lists which are ephemeral.

    However it's more difficult to write in a notebook on the move. As a hybrid paper/iPhone approach I'm using Erica Sadun's 'To Do' iPhone application to capture items on the move which I can easily do one handed while walking (e.g. while changing trains). These then get added to the AF list at some convenient point later (e.g. when I've sat down on the next train). In GTD terms it's another collection device.

    I don't care about syncing to the desktop as they end up on paper anyway. None of this uses the iPhone's data access features so could be done with an iPod Touch which is much cheaper.

    I keep 'To Do' on the task bar for easy access along with phone, text and mail. I haven't done an extensive survey but the things I like about it are:

    Nice large font. Items are only in here temporarily so I won't have many of them.

    No spelling correction which means I can use abbreviations or shorthand without worry about them being changed. Spelling mistakes usually don't matter anyway.

    A nice loud red number overlay showing how many items there are in the list.

    Items can be added in a minimal number of clicks

    Items can be easily deleted one at a time or en masse (though a delete all would be useful).

    It's very simple and I don't forget how to use it when I haven't used it for a while

  • Hi, Mike. I'm on day 3 of AF using Tadalist and I love it. I don't mind paper, but it bothers me that I can't have it on my person at all times. When a task comes to mind, I have to run to where my notebook is to write it down. I've usually forgotten by the time I get there! And copying items to my list later seems tedious. I just add items to my last list via IPhone. Very fast and no re-writing. Unlike paper, I can clearly see the number of outstanding items on all my lists on www.tadalist.com. Then inside each list, I see only the undone items on top. This is much easier on my eyes than searching through a paper list with lots of crossed off items. Re-entering repeating items in the list is so simple online as the computer suggests the task as you begin to type. On my Mac, I keep my Tadalist on my bookmark's toolbar, so I can access it quickly as I complete tasks. I really love it and don't think you have to be an IPhone user to enjoy using Tadalist. If you work with an Internet-connected computer or mobile device throughout the day, it's worth checking out! Just so you know, I'm not affiliated with them in any way! :-)

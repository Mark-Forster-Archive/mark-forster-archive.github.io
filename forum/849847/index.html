Morning Procedure

  • I've written out the steps needed each morning as an aide memoire. I'm sure after a few weeks I won't need this but here it is:

    1. Highlight yesterday's unactioned tasks on notice
    2. Draw a line below task(s) on notice for review
    3. Date tasks actioned yesterday
    4. Record today's date
    5. Delete and re-enter "change to forward mode" if not actioned yesterday
    6. Start in reverse mode

    The system is working like a dream after day one. I'm getting so much done.

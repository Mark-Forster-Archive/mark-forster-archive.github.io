IMPORTANT NOTICE

  • Because spam is becoming a major problem on this discussion group, I have decided to make it membership only. This will not cost anything, but means that you must register before you can use the site, and will have to sign on when you want to make a posting.

    If you wish to register for the site, please email me at mf@markforster.net with the following details:

    1) Account login name (alphanumeric characters only - minimum 4)

    2) Display Name (the name you wish your postings to be displayed under - may contain spaces)

    3) Password (6-25 characters)

    I recommend if you were previously a frequent poster to the site that you use the same Display Name to ensure continuity.

  • Some recurring tasks, I re-add on the +7 page, some I re-add on the +30 page. I think the main difference is whether I think of the task as completed for today or not.

    For example - 'check email' isn't completed until I go home, so it gets re-added on +7 as a 'continuing task'. But 'update status' needs to be done twice a week, but it's done for today so it gets re-added on +30 as a 'new task' (with a due date of Thursday)

  • I have repeating tasks set up in OmniFocus to handle this (that's all I use the program for now). Some of them repeat every 4 hours, some every week, month, etc. The Start Date feature keeps them hidden until it's actually time to do them.

  • I find I just can't stand not to check my email several times a day. So I have "email" entered each day and put a checkmark next to the entry each time I look at email. That way I'm aware of spending time on it and don't go too overboard.

    I also use checkmarks for a project that involves several steps that is time-sensitive and/or I expect to be finished it in a few days. That way I don't have to keep moving it ahead (MF says only one 7 day re-entry ((or is that just for monthly expires?)), and I've given myself credit by placing a checkmark next to it. By the time it's done I might have 2-3 checkmarks then cross it off. A little diversion from the system I suppose.

    In fact, this might work better for me because I have so little time to "work the list", and I stay focused on a smaller area in the daily diary.

  • I use an approach similar to Geoff.

    In ListPro I have 3 tasks, Daily Routines, Weekly and Monthly. Each has sub-tasks.

    At the end of every day, for example, I note which sub-tasks have not been addressed, determine the reason if I can (spending too much time on this forum!?!), then un-check the done tasks and change the diary date to the next day.

    Like Bev P, I mark the Check emails task as done the first time, then write it again in ink (the DWM list is printed every morning, 2 x A5 pages on one side of an A4 sheet, thus I get the best of both electronic and paper worlds) below the last item on the list (printed or written).

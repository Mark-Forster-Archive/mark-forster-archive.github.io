Small items

  • According the descriptions for the autofocus systems you should go through the list until you find a project that is "workable" and work on it. But how do you handle the minor stuff that sometimes is critical to move a project forward?

    Say my list looks like this:

    1. Write small statistics program
    2. Contact insurancecompany for a new deal
    3. Mount shelf in livingroom

    Now sitting at work going through this list I can do 1 and 2 while 3 has to be done at home. But actually to get 3 done I need to buy nails so to get going I need a trip to the hardwarestore.

    So what do I do with this as it not really workable when I see it but still needs to be planned so I go to the hardwarestore when ever I can?
    25 February 2011 at 10:34 | Juman
  • Add a task "nails hardwarestore" at the end of your list and move it to column 2 when it stands out (because you really want to get the damn shelf mounted, or you realize you have some time to go shopping) as a task to be done today (when you travel from work to home for example). Re-enter it in column 2 of the next page until it is done.

    That's what I would do, I think. Someone else?

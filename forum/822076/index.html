AutoFocus and workload

  • I've been using AutoFocus since April, and AutoFocus 2 since last Saturday. So far I really love AF2, addresses all the issues I had with AF.

    One thing that's bothering me more and more though: the dismissal process in both AF and AF2 makes me more acutely aware of the fact that my workload is too high. I guess I should be happy about that, but I've been telling myself that I really can handle all these projects, that I'm just not efficient enough. Apparently it's time for a reality check...

    Anybody else have this experience?

  • Yep, I have this all the time - it's like there's too many interesting things in life to dip your toe into, so you end up trying to eat everything at once and don't really enjoy any of it. (How's that for a mixed metaphor?)

    I've found an 80/20 principle approach quite useful on occasion...to get to the real core projects that will make a difference, and keep an overview of what is really important.

  • Mel,

    I find that to be a tricky issue. The basic thing that caused me so much trouble with AF2 was that "someday/maybe" items were getting on to my list and clogging it up. (That oversimplifies it, but it is about right.) I found that if I get too many things on the list that I'm not committed to doing then I get stuck. My solution was to ask myself, every time I add something, if I REALLY REALLY INTEND to do it this week. If not, I immediately dismiss it ... which makes it a "someday/maybe" that I can come back to later. But it does get it out of the road.

    But your question was about "shoulds". Yes, but only if you WILL ... is my answer.

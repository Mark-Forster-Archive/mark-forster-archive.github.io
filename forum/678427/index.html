Where to start each day

  • I understand that each day I should start my AF list where I left it the last time I was working. Am I the only one finding it hard to remember where I was up to in all the pages? What systems to people have for knowing where to continue from? And do you do the first scan on the page you were on the day before or start with scanning the next page? I ask this because, although I know it 's not quite how it should work, I do seem to get bogged down in one page for days at a time. Thanks.

  • This raises an interesting question.

    Do you complete your current task at the end of the day?

    If not, the "committed dot" tells you where you are - there is only ever one incomplete task with a "committed dot". A bookmark helps significantly!

    On an electronic system it's relatively straightforward: using your "current task" marker, you mark the next task when you close for the day, and clear it the next morning.

    But you really should be doing multiple passes every day. If you're getting stuck on a single page, something is seriously wrong. So you should be able to restart at the beginning each day. It might be worth walling off an hour or two where you consciously limit the time you allow yourself for each task (to 5 or ten minutes, say).

  • I use little Post It Notes as tabs. So I have one of one color on the current active page, and one of another color on the last page, which makes flipping to the end and adding new stuff easy. And moving a Post It tab is also easy. You just pull up the tab, flip the notebook page, and slap the tab back down. (Neatly, of course.)
